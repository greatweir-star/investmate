# 用户须知

用于记录用户在使用 InvestMate 前需要了解的基础事项。

## 1. 产品用途

InvestMate 用于信息整理、结构化分析和历史记录管理。

## 2. 用户责任

用户需要结合自身情况、数据来源和个人目标进行独立判断。

## 3. 数据说明

系统分析依赖数据源。若数据缺失、过旧或异常，页面应明确提示。

## 4. AI 说明

AI 解释基于结构化结果生成，可能存在不完整或需要人工复核的情况。

## 5. 隐私说明

MVP 阶段如需录入持仓，默认仅用于当前产品分析和记录展示。

## 6. 页面展示位置

用户须知应在以下位置出现：

- 首次进入页面；
- 设置页；
- 分析结果底部；
- 历史详情页。
