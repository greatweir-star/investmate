# 用户访谈脚本

## 目标

了解用户是否能理解 InvestMate 的核心价值，以及 MVP 哪些地方需要优化。

## 访谈时长

建议 20 到 30 分钟。

## 开场

感谢你参与 InvestMate 内测。我们希望了解你真实的使用感受，没有标准答案。

## 问题一：使用背景

1. 你平时会使用哪些工具做信息整理？
2. 你通常如何判断一个标的是否值得继续研究？
3. 你是否有记录和复盘的习惯？

## 问题二：首次体验

1. 首页是否容易理解？
2. 你是否知道下一步应该点击哪里？
3. 哪个模块最吸引你？
4. 哪个模块让你困惑？

## 问题三：分析结果

1. 摘要是否清楚？
2. 评分是否容易理解？
3. 解释内容是否有帮助？
4. 你还希望看到哪些信息？

## 问题四：持仓录入

1. 你是否愿意手动录入组合？
2. 哪些字段让你觉得麻烦？
3. 你是否担心隐私？
4. 什么提示会让你更放心？

## 问题五：长期使用

1. 什么场景下你会再次打开？
2. 什么功能会让你持续使用？
3. 什么问题会让你放弃使用？

## 结束问题

如果只能保留一个功能，你希望保留哪个？为什么？

## 记录方式

记录用户原话、困惑点、重复出现的问题和明确需求。
