# PRODUCT BLUEPRINT

## Product Name

InvestMate

## One-line Positioning

An AI investment copilot that provides a transparent second opinion before important investment decisions.

## Target User

Retail investors who already invest or plan to invest but struggle with uncertainty, information overload, position sizing, and emotional decisions.

## Core Pain Points

1. Too much information, too little decision clarity.
2. No personalized view based on their risk profile and portfolio.
3. Frequent emotional mistakes: chasing, panic selling, over-concentration.
4. Lack of a trusted second opinion before taking action.

## Value Proposition

InvestMate helps users answer:

- Should I buy this asset now?
- Should I sell or hold?
- Is my current portfolio too risky?
- What exposure level is reasonable today?
- What evidence supports or weakens this decision?

## MVP Product Modules

### 1. Investment DNA

User onboarding profile defining risk, horizon, capital size, investment style and constraints.

### 2. Daily Decision

A concise daily market-aware decision support card.

### 3. Second Opinion

User enters an asset and intended action. InvestMate returns an AI second opinion.

### 4. Portfolio Review

User inputs holdings. InvestMate evaluates concentration, style bias, risk and alignment.

### 5. Decision History

Every AI output is saved for user review and future trust-building.

## Product Philosophy

Decision first, explanation second, evidence third.

Users should not be forced to read a report before understanding the recommendation.

## Commercial Model Hypothesis

Free:

- General Daily Decision
- Limited Second Opinion

Paid:

- Personalized portfolio review
- Full decision history
- Watchlist monitoring
- Higher frequency analysis
- Custom Investment DNA

## MVP Success Metrics

- User completes Investment DNA onboarding.
- User asks at least one Second Opinion.
- User returns before another trade decision.
- User saves or revisits a decision history item.
- User says the product helped them avoid or improve a decision.
