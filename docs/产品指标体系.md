# 产品指标体系

用于记录 InvestMate MVP 阶段需要观察的核心指标。

## 1. 核心指标

用户是否在需要整理信息和回看记录时主动打开 InvestMate。

## 2. 激活指标

- 首次访问；
- 完成一次标的分析；
- 查看完整结果；
- 保存历史记录。

## 3. 深度使用指标

- 录入持仓；
- 完成持仓分析；
- 查看历史详情；
- 修改用户偏好。

## 4. 留存指标

- 次日再次访问；
- 七日内再次访问；
- 回看历史记录；
- 重复使用同一功能。

## 5. 反馈指标

- 提交反馈；
- 参加访谈；
- 提出建议；
- 主动分享。

## 6. 每周复盘

每周整理：

- 新增用户；
- 完成首次体验人数；
- 持仓录入人数；
- 历史回看人数；
- 用户反馈摘要；
- 下周优化重点。
