# 冷启动计划

## 1. 目标

用最小成本找到第一批真实用户，验证 MVP 是否值得继续开发。

## 2. 阶段一：熟人内测

目标人数：20 人以内。

方式：

- 朋友圈邀请；
- 一对一私聊邀请；
- 小范围微信群体验。

目标：发现明显体验问题。

## 3. 阶段二：垂直社群测试

目标人数：50 到 100 人。

方式：

- 投资学习社群；
- AI 工具社群；
- 个人知识星球或微信群。

目标：验证非熟人用户是否能理解产品。

## 4. 阶段三：公开内容获客

渠道：

- 小红书；
- 公众号；
- 视频号；
- 即刻；
- 知乎。

内容形式：

- 产品演示；
- 使用案例；
- 复盘方法；
- 工具思路分享。

## 5. 冷启动转化路径

用户看到内容，进入产品，完成一次分析，留下反馈或联系方式。

## 6. 成功信号

- 用户主动转发；
- 用户提出明确改进建议；
- 用户愿意再次体验；
- 用户愿意参加访谈。

## 7. 迭代节奏

每周整理一次反馈，每两周发布一个小版本。
