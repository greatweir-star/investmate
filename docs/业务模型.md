# 业务模型

用于记录 InvestMate 的业务结构、用户价值和阶段目标。

## 1. 用户价值

InvestMate 的核心价值不是提供更多信息，而是帮助用户整理信息、形成结构化理解、保存历史记录。

## 2. 用户分层

### 免费体验用户

目标：完成首次体验，理解产品价值。

能力：

- 查看市场状态；
- 完成有限次数标的分析；
- 查看基础历史记录。

### 深度使用用户

目标：验证个性化能力是否有长期价值。

能力：

- 持仓结构分析；
- 用户投资 DNA；
- 历史记录；
- 周期性复盘。

### 专业使用用户

目标：验证更高频、更系统化的使用场景。

能力：

- 多组合；
- 关注列表；
- 批量分析；
- 报告模板。

## 3. 价值闭环

```text
首次体验
  ↓
理解结构化分析
  ↓
录入个人信息
  ↓
获得更贴近个人场景的结果
  ↓
保存历史
  ↓
形成复盘习惯
```

## 4. 阶段目标

MVP 阶段只验证：

1. 用户是否愿意输入标的；
2. 用户是否愿意录入持仓；
3. 用户是否愿意查看历史记录；
4. 用户是否愿意再次使用；
5. 用户是否愿意参加访谈。

## 5. 暂缓事项

第一阶段暂缓复杂商业系统、复杂权限系统和大规模运营系统。
