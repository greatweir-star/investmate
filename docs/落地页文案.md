# 落地页文案

## 1. 首屏标题

InvestMate：你的 AI 投研辅助副驾驶

## 2. 副标题

在做重要判断前，用 AI 帮你整理数据、结构化信息、发现需要关注的问题。

## 3. 核心卖点

### 3.1 结构化分析

不只是长篇文字，而是摘要、评分、关键因素和解释。

### 3.2 个性化视角

结合你的投资偏好和持仓结构，提供更贴近个人情况的分析。

### 3.3 历史可回看

每次分析都可以保存，方便之后复盘。

## 4. 适合谁

- 希望减少信息焦虑的人。
