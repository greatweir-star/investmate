# 发布

记录产品发布相关事项。

## 1. 目标

让第一批用户知道产品，并完成首次体验。

## 2. 发布主题

- 产品是什么；
- 首页怎么用；
- 标的分析怎么用；
- 持仓页面怎么用；
- 历史记录怎么用；
- 如何反馈问题。

## 3. 发布渠道

- 公众号；
- 小红书；
- 知乎；
- 视频号；
- 社群；
- 朋友圈。

## 4. 发布节奏

每周至少三次更新：

- 一次产品演示；
- 一次使用说明；
- 一次反馈总结。

## 5. 发布检查

发布前检查：

- 页面是否可访问；
- 核心流程是否可完成；
- 文案是否清楚；
- 反馈入口是否可用；
- 用户须知是否可见。

## 6. 发布后跟踪

发布后记录：

- 访问人数；
- 首次体验人数；
- 反馈数量；
- 再次访问人数；
- 主要问题。
