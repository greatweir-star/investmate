# 使用说明文案

用于记录产品中的基础说明文案。

## 1. 首次进入

欢迎使用 InvestMate。它会帮助你整理数据、查看结构化分析，并保存历史记录。

## 2. 首页说明

首页展示当前市场概览、快速分析入口和最近记录。

## 3. 标的分析说明

输入股票、ETF 或指数代码，系统会生成结构化分析结果。

## 4. 持仓分析说明

你可以手动录入持仓，系统会帮助你查看组合结构。

## 5. 历史记录说明

历史记录会保存系统过去生成的分析，方便你后续回看。

## 6. 设置说明

设置页用于配置数据源、模型服务和个人偏好。

## 7. 通用说明

系统输出用于辅助理解和信息整理，最终判断仍由用户完成。
