# 增长

## 1. 目标

记录产品早期增长相关事项。

第一阶段增长目标不是追求大规模曝光，而是找到愿意真实使用、愿意反馈、愿意参与内测的种子用户。

## 2. 渠道

优先渠道：

- 公众号；
- 小红书；
- 知乎；
- 视频号；
- 社群；
- 朋友圈。

## 3. 内容方向

### 3.1 产品演示

展示首页、标的分析、持仓分析和历史记录的真实使用流程。

### 3.2 使用方法

说明用户如何通过 InvestMate 整理信息、查看摘要、保存记录。

### 3.3 版本更新

每次小版本更新后，说明新增能力和优化点。

### 3.4 用户反馈

整理内测用户高频问题和改进方向。

## 4. 冷启动路径

```text
熟人内测
  ↓
社群测试
  ↓
公开内容
  ↓
等待名单
  ↓
重点用户访谈
```

## 5. 转化路径

```text
看到内容
  ↓
进入产品
  ↓
完成一次体验
  ↓
查看结果
  ↓
留下反馈
  ↓
加入内测
```

## 6. 指标

- 访问；
- 首次体验；
- 反馈；
- 再次访问；
- 内测申请；
- 访谈完成。

## 7. 周节奏

每周至少完成：

- 一次产品演示内容；
- 一次方法类内容；
- 一次用户反馈整理；
- 一次产品小迭代总结。

## 8. 判断标准

如果用户愿意完成首次体验并留下反馈，说明冷启动渠道有效。

如果用户愿意再次使用并参与访谈，说明产品价值有进一步验证空间。
