# 产品边界说明

用于说明 InvestMate 的产品定位和能力边界。

## 定位

InvestMate 是 AI 投研辅助工具。

它帮助用户整理信息、查看结构化分析、记录历史结果，并建立个人研究流程。

## 不做的事情

- 不执行用户操作
- 不连接真实账户
- 不托管资产
- 不承诺结果
- 不制造紧迫感
- 不替代用户判断

## 默认说明

InvestMate 输出仅用于信息整理和辅助理解。用户需要结合自身情况、数据来源和风险承受能力进行独立判断。

## 页面展示要求

以下位置需要展示边界说明：

- 首次进入产品
- 分析结果底部
- 设置页面
- 历史详情页面

## Agent 要求

Agent 输出必须包含前提、数据日期和不确定性说明。
