# 用户投资 DNA

## 1. 为什么需要用户投资 DNA

同一个市场环境下，不同用户需要关注的问题并不相同。

一个本金较小、投资期限较长、风险承受能力较高的用户，和一个本金较大、接近退休、偏好稳健收益的用户，不应该得到完全一样的辅助分析。

因此，InvestMate 不应该只分析市场，还应该理解用户。

用户投资 DNA 是系统为用户建立的长期画像，用于让 AI 输出更符合用户自身情况。

## 2. 投资 DNA 的组成

### 2.1 基础信息

- 年龄段；
- 投资经验；
- 可投资资金规模；
- 是否有稳定现金流；
- 是否有短期资金使用需求。

### 2.2 风险偏好

- 保守型；
- 稳健型；
- 平衡型；
- 积极型；
- 高风险型。

风险偏好不直接等同于风险能力。系统需要区分：

- 用户愿意承担多少波动；
- 用户实际能承受多少损失；
- 用户在亏损时是否容易情绪化。

### 2.3 投资目标

- 资产保值；
- 稳健增值；
- 长期成长；
- 现金流；
- 学习投资；
- 阶段性配置。

### 2.4 投资期限

- 1 个月以内；
- 1 到 6 个月；
- 6 到 12 个月；
- 1 到 3 年；
- 3 年以上。

### 2.5 最大可接受回撤

用户可以选择自己心理上可以承受的最大账户回撤：

- 5% 以下；
- 5% 到 10%；
- 10% 到 20%；
- 20% 到 30%；
- 30% 以上。

### 2.6 资产偏好

- A 股个股；
- ETF；
- 指数基金；
- 港股；
- 美股；
- 债券；
- 黄金；
- 现金类资产。

MVP 阶段只支持 A 股和 ETF，但用户偏好要提前预留。

### 2.7 行为偏好

- 喜欢长期持有；
- 喜欢波段操作；
- 容易追热点；
- 容易恐慌；
- 经常满仓；
- 交易频率较高；
- 偏好分散配置。

行为偏好可以通过问卷获取，也可以在后续版本中通过历史行为推断。

## 3. 投资 DNA 的用途

### 3.1 调整风险提示强度

同一个标的，如果波动较大，对保守型用户需要更明显提示，对积极型用户则可以强调波动来源和适配条件。

### 3.2 调整分析侧重点

长期用户更关注基本面和估值。

短期用户更关注趋势、流动性和波动。

稳健用户更关注回撤和行业集中度。

### 3.3 调整语言风格

对新手用户，系统应减少术语，多解释原因。

对专业用户，系统可以显示更多指标和细节。

### 3.4 识别组合不匹配

如果用户是保守型，但组合集中在高波动成长行业，系统应该提示投资 DNA 与组合之间存在偏差。

## 4. MVP 问卷字段

第一版只需要 8 个问题：

1. 你的主要投资市场是什么？
2. 你的投资经验大概多久？
3. 你更偏好哪类资产？
4. 你的投资周期通常多长？
5. 你可以接受的最大账户回撤是多少？
6. 你更看重稳健还是成长？
7. 你通常多久调整一次持仓？
8. 当账户下跌时，你通常会怎么做？

## 5. 数据结构建议

```json
{
  "risk_level": "balanced",
  "investment_horizon": "1-3y",
  "max_drawdown_tolerance": 0.15,
  "asset_preferences": ["a_share", "etf"],
  "style_preference": "balanced_growth",
  "trading_frequency": "monthly",
  "behavior_tags": ["容易受热点影响", "偏好分散配置"],
  "experience_level": "intermediate"
}
```

## 6. 后续扩展

未来可以加入：

- 用户历史操作行为；
- 用户关注列表；
- 用户复盘记录；
- 用户对 AI 输出的反馈；
- 用户真实持仓变化；
- 用户风险承受能力动态调整。

投资 DNA 不应该一次填写后永久不变，而应该随着用户行为和反馈持续更新。
