# 能力分层

## 1. 目标

能力分层用于明确 InvestMate 哪些能力免费体验，哪些能力适合作为高级能力验证。

## 2. 免费体验层

目标：让用户快速理解产品价值。

能力：

- 查看首页；
- 生成一次持仓体检报告；
- 查看示例报告；
- 保存少量历史记录。

## 3. 深度体验层

目标：验证用户是否愿意持续使用。

能力：

- 更多报告次数；
- 完整历史记录；
- 用户投资 DNA；
- 周度复盘；
- 反馈跟踪。

## 4. 专业使用层

目标：验证更系统化的长期使用场景。

能力：

- 多组合管理；
- 关注列表；
- 多期报告对比；
- 导出报告；
- 更完整的数据解释。

## 5. 第一阶段边界

MVP 不急于实现复杂商业系统。

第一阶段只需要在产品中预留：

- 报告次数限制；
- 历史记录上限；
- 周度复盘入口；
- 内测申请入口。

## 6. 转化点

转化不应出现在用户还没体验价值之前。

建议转化点放在：

- 用户生成报告之后；
- 用户尝试保存更多历史时；
- 用户想生成周度复盘时；
- 用户想管理多个组合时。

## 7. 验收标准

- 页面能区分基础能力和高级能力；
- 高级能力入口不影响基础体验；
- 用户能先体验再决定是否深入使用。
