# 前端组件清单

本文档记录 MVP 阶段需要实现的前端组件，供 Codex 开发时使用。

## 1. QuickAnalyzeBox

首页快速入口。用户输入代码或名称后跳转到标的分析页。

字段：defaultKeyword、placeholder、onSubmit。

验收：空输入不可提交；提交后跳转到 `/assets`。

## 2. RecentHistoryList

最近记录列表。展示最近生成的分析记录。

字段：analysis_id、analysis_type、target_name、summary、created_at、data_date。

验收：可展示列表；无记录时显示空状态。

## 3. AssetSearchInput

标的搜索框。支持输入代码或名称。

字段：keyword、suggestions、loading、onSearch、onSelect。

验收：可搜索、可选择、无结果有提示。

## 4. AssetScorePanel

标的评分面板。展示趋势、估值、流动性、波动、用户匹配度。

字段：trend、valuation、liquidity、volatility、fit_to_user。

验收：评分为 0 到 100；缺失时显示空值。

## 5. FactorList

因素列表。分组展示支持因素和关注因素。

字段：keyFactors、watchFactors、title。

验收：支持空列表；文案为中文。

## 6. ExplanationPanel

解释面板。展示摘要、详细解释、置信度和数据日期。

字段：summary、explanation、confidence、dataDate。

验收：中文展示；数据日期可见。

## 7. PositionEditor

持仓编辑器。用于手动录入组合内容。

字段：asset_code、asset_name、weight、cost_price、note。

验收：可新增、编辑、删除；比例有校验。

## 8. PortfolioPanel

组合结构面板。展示集中度、行业分布、风格分布和用户匹配度。

字段：concentration、industryExposure、styleExposure、fitToUser、summary。

验收：可以展示组合摘要和结构信息。

## 9. HistoryTable

历史记录表格。展示历史分析列表。

字段：created_at、analysis_type、target_name、summary、confidence、data_date。

验收：按时间倒序展示；可以进入详情。

## 10. SettingsForm

设置表单。用于配置用户投资 DNA 和基础参数。

字段：risk_level、investment_horizon、max_drawdown_tolerance、asset_preferences、experience_level。

验收：可以保存；敏感字段不明文展示。
