# 组件规格：QuickAnalyzeBox

## 1. 组件目标

QuickAnalyzeBox 是首页的快速分析入口，让用户可以直接输入标的代码或名称，并跳转到标的分析页。

## 2. 使用页面

- 首页 `/`

## 3. 输入 Props

- defaultKeyword：默认输入内容，可选；
- placeholder：输入提示文案；
- onSubmit：提交回调。

## 4. UI 元素

- 输入框；
- 分析按钮；
- 简短说明文案。

## 5. 交互规则

1. 用户输入为空时，按钮不可提交；
2. 用户点击按钮后，跳转到 `/assets?keyword=输入内容`；
3. 回车键也可以触发提交；
4. 输入内容前后空格需要清理。

## 6. 空状态文案

```text
输入股票、ETF 或指数代码，开始一次结构化分析。
```

## 7. 错误状态

如果输入过短：

```text
请输入有效的代码或名称。
```

## 8. 验收标准

1. 可以输入内容；
2. 可以点击提交；
3. 可以跳转到标的分析页；
4. 空输入不能提交；
5. 所有文案为中文。
