# 页面规格：设置

## 1. 页面目标

设置页用于配置 InvestMate MVP 的基础运行参数。

页面回答的问题是：

> 系统使用哪些数据、哪些模型，以及如何根据我的投资偏好工作？

## 2. 路由

```text
/settings
```

## 3. 页面组成

页面包含以下组件：

1. AppHeader；
2. DataSourceSettings；
3. AIServiceSettings；
4. InvestmentDNAForm；
5. DataSyncStatus；
6. SaveSettingsButton。

## 4. 数据依赖

### 4.1 获取设置

接口：

```text
GET /api/settings
```

### 4.2 保存设置

接口：

```text
POST /api/settings
```

### 4.3 获取数据同步状态

接口：

```text
GET /api/admin/sync/status
```

## 5. 设置字段

### 5.1 数据源设置

- tushare_token_status；
- latest_data_date；
- sync_status。

注意：前端不展示完整 Token，只展示是否已配置。

### 5.2 AI 服务设置

- model_provider；
- model_name；
- api_key_status。

注意：前端不展示完整 Key，只展示是否已配置。

### 5.3 用户投资 DNA

- risk_level；
- investment_horizon；
- max_drawdown_tolerance；
- asset_preferences；
- trading_frequency；
- experience_level。

## 6. 用户流程

```text
进入设置页
  ↓
查看数据源状态
  ↓
配置或更新必要参数
  ↓
填写投资 DNA
  ↓
保存设置
  ↓
返回首页或继续同步数据
```

## 7. 安全展示规则

1. 不在前端明文展示完整 Token 或 Key；
2. 只显示“已配置”或“未配置”；
3. 保存成功后不回显敏感字段；
4. 敏感字段修改需要重新输入完整值。

## 8. 空状态

未配置数据源时显示：

```text
尚未配置数据源。配置后才能同步基础数据。
```

未配置投资 DNA 时显示：

```text
请完善投资偏好，以便系统生成更适合你的分析结果。
```

## 9. 验收标准

1. 可以展示当前设置状态；
2. 可以保存用户投资 DNA；
3. 可以展示数据同步状态；
4. 不明文展示敏感字段；
5. 保存成功和失败都有中文提示。
