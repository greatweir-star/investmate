# InvestMate 🧠📈

**Investment Copilot — Let AI help you invest smarter.**

InvestMate 是一个 AI 驱动的投资助手项目，旨在通过自然语言交互和多 Agent 协作，帮助投资者完成行情分析、财报解读、组合管理、风险预警、宏观研究等投资决策工作流。

## 项目结构

```
investmate/
├── docs/          # 项目文档与设计稿
├── specs/         # 需求规格与接口设计
├── agents/        # AI Agent 定义与编排
├── prompts/       # 提示词库与模板
├── backend/       # 后端服务
├── frontend/      # 前端界面
├── scripts/       # 工具脚本
└── examples/      # 使用示例与教程
```
