# MILESTONE 01 — Local MVP

## Objective

Create a usable local MVP that demonstrates the core user experience of InvestMate.

## Acceptance Criteria

- User can create Investment DNA.
- User can view Daily Decision dashboard.
- User can ask for a Second Opinion.
- User can input a portfolio and receive a review.
- User can view decision history.
- All AI outputs follow structured schema.
- Compliance disclaimer is visible.

## Excluded

- Real-time data
- Live brokerage/trading
- Mobile app
- Payment
- User community

## Suggested Timeline

Day 1-2: project scaffold and UI shell  
Day 3-4: database and forms  
Day 5-6: mocked decision engine  
Day 7: local demo and iteration
